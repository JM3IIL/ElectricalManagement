直流送電
- 海底ケーブル
- 無効電力による送電ロスがない(長距離送電可能)
- 電線2本でよい(建設費低減)
- 安定度の問題がない(大容量の送電)
- 同じケーブルなら交流と比べ√2倍の高圧で送電可能
- 最大電圧と実効値が同じ(絶縁距離を短くできる)
- 充電電流の補償が不要(交流送電では、分路リアクトルにより、対地静電容量による進相無効電力を補償)
- 電流零点がないため、遮断しにくい
- 交直変換装置が必要
- 変換装置から高調波が発生
- 大地帰路方式で電食が発生しやすい

故障点位置評定
- マーレーループ法：ホイートストンブリッジの原理：ケーブル亘長：健全相がない場合、断線事故の場合、適用できない
- パルスレーダ法：事故点での反射パルスを検知：パルス電圧の伝搬速度：地絡事故、断線事故、標定可能。測定誤差が生じやすい

- 有効電力： P [W]
- 皮相電力： S [VA]
- 無効電力： Q [var]
力率： $\cos\theta = \frac{P}{S} = \frac{P}{\sqrt{P^2+Q^2}}$


【単相】
- 電圧： V [V]
- 電流： I [A]
- $P = VIcosθ [W]$
- $Q = VIsinθ [var]$

【三相】
- 線間電圧： V [V]
- 線電流： I [A]
- $P = \sqrt{3}VIcosθ [W]$
- $Q = \sqrt{3}VIsinθ [var]$
- $S = \sqrt{3}VI [V⋅A]$

【三角関数基礎】
- $sin^2θ + cos^2θ = 1$
- $sinθ = \sqrt{1 - cos^2θ}$
- $tanθ = \frac{sinθ}{cosθ}$
